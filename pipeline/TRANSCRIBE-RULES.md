# Per-page transcription rules (v2, 2026-07-10 — Sonnet pipeline)

1. VERBATIM: reproduce the text exactly as printed — do not modernize spelling, punctuation, or correct apparent errors. Preserve curly quotes/apostrophes as printed.
2. DEHYPHENATE within the page: a word split across a line break joins into one word. Keep genuine compound hyphens.
3. REFLOW: one line per paragraph. Preserve paragraph boundaries.
4. OMIT the running head and the page folio (number).
5. HEADINGS: ONLY if a heading is printed in the body of THIS page: chapter → "## ", section → "### ", small caps → title case. NEVER infer or invent a heading from context — a page that is mid-chapter body text gets NO heading.
6. PAGE-BOUNDARY HYPHEN: if the page's last printed word is hyphenated (continues on the next page), keep the trailing hyphen as-is. If the page BEGINS mid-word or mid-sentence, transcribe exactly as printed — the stitcher joins pages.
7. FOOTNOTES: transcribe at the end of the page block under a line `FOOTNOTES:`, each as `^N text`. Keep the in-text markers as `[^N]`.
8. Ignore ink marks, checkmarks, library stamps, marginalia.
9. Uncertain reading → [?word].
